# CrossFit Córdoba Shop

Proyecto de diseño web para el módulo de **Implantación de Aplicaciones Web (ASIR)**. Consiste en la creación de una tienda online estructurada en tres páginas estáticas con interactividad básica.

## Características del Proyecto

*   **Página de Inicio (`index.html`):** Portada principal con banner, presentación de la camiseta destacada e introducción a la comunidad.
*   **Página de Producto (`producto.html`):** Vista en detalle del artículo con galería de imágenes intercambiables, selección de talla y simulación funcional de añadir al carrito.
*   **Formulario de Contacto (`contacto.html`):** Formulario de atención al cliente con validación de campos obligatorios.
*   **Diseño Responsivo:** Adaptación a pantallas de dispositivos móviles mediante el uso de Media Queries (`@media`).

## Tecnologías Utilizadas

*   **HTML5:** Estructura semántica de las páginas.
*   **CSS3:** Maquetación mediante Flexbox, gradientes, transiciones y diseño adaptable (Responsive Design).
*   **JavaScript (Vanilla JS):** Manipulación del DOM para la validación de formularios, cambio de imágenes en la galería y alertas del carrito de la compra.
*   **Google Fonts:** Implementación de las tipografías *Oswald* para encabezados y *Roboto* para el cuerpo de texto.
